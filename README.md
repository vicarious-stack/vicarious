# Vicarious

A literary‑fiction project blending neuroscience, psychology, and narrative storytelling.

- **Read online**: The web reader (`index.html`) renders the book with a page‑fold transition, optimized for mobile devices.
- **Repository**: Contains the full manuscript (`chapter_01.md` … `chapter_12.md`, `epilogue.md`, `appendices.md`), supporting scripts (`assemble.py`, `word_count.py`, `validate.py`), and the web reader implementation.
- **How to view**: Open `index.html` in a browser (mobile or desktop). The pages fold like a physical book.

## Getting Started
```bash
# Clone the repo
git clone https://github.com/vicarious-stack/vicarious.git
cd vicarious
# Install dependencies (none needed for pure HTML/JS)
# Open the reader
open index.html   # or double‑click the file in Explorer
```
